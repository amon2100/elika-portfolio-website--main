# Elika Information and Security Technologies

Modern website built with Vite + React + TypeScript + Tailwind CSS.

Created by Kabwigu Amon Eliya.

## Overview

This repository contains the source code for the Elika Information and Security Technologies website. It showcases services in IT infrastructure, cybersecurity, CCTV installation, access control, and smart security systems.

## Tech Stack

- Vite
- React 18
- TypeScript
- Tailwind CSS

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

The app runs at http://localhost:5173/

## Production Build

Create an optimized build into the `dist/` folder:

```bash
npm run build
```

Optionally preview the production build locally:

```bash
npm run preview
```

## Deployment

The `dist/` folder can be deployed to platforms like Vercel or Netlify. Configure your project to use `npm run build` as the build command and `dist/` as the publish directory.
